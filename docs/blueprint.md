# **App Name**: LankaHorizon Travels

## Core Features:

- Homepage Hero Section: Showcase a captivating hero image of Sri Lanka with a clear headline and a call-to-action button that redirects to WhatsApp.
- Tours/Packages Display: Display travel packages with details such as destination, duration, and short description.
- WhatsApp Booking: Implement WhatsApp Click-to-Chat to allow users to book tours directly via WhatsApp.
- Custom Itinerary Form: A form for users to submit their travel preferences which then redirects to WhatsApp with the form data.
- Floating WhatsApp Button: Implement a floating WhatsApp button for instant access to booking inquiries.
- Destination Highlights: Use an AI tool to create short, engaging descriptions for each tour destination.
- SEO Optimization: Implement SEO-friendly meta tags and image optimization.

## Style Guidelines:

- Primary color: Lush Green (#3CB371), evoking the tropical landscapes of Sri Lanka.
- Background color: Soft, desaturated white (#F5F5F5), providing a clean and modern backdrop.
- Accent color: Ocean Blue (#4682B4), inspired by the coastal beauty of Sri Lanka, for CTAs and highlights.
- Body and headline font: 'Inter', a grotesque-style sans-serif with a modern look, for clear and readable text.
- Use icons that represent Sri Lankan culture, nature, and travel destinations.
- Implement a mobile-first, responsive layout optimized for all devices.
- Use subtle animations on the page to enhance user experience without being distracting.
